## 1. GitHub Student Developer Pack

The Pack is the gateway service. Apply first, then redeem individual benefits.

**How to apply:** https://education.github.com/pack
- Requires `.edu.tw` email added to GitHub account
- Upload student ID or enrollment certificate
- Approval: usually 3-7 days (sometimes instant for `.edu.tw`)

### Pack Benefits

| ID | Service | What You Get | Value | URL |
|----|---------|-------------|-------|-----|
| copilot | GitHub Copilot | AI code assistant, free during enrollment | ~$120/yr | https://github.com/settings/copilot |
| github-pro | GitHub Pro | Unlimited private repos, 3000 Actions min/mo, 180h Codespaces | ~$48/yr | (auto-enabled) |
| gitkraken | GitKraken Pro | Professional Git GUI client | ~$60/yr | https://www.gitkraken.com/github-student-developer-pack |

| 1password | 1Password | Personal account free 1 year, SSH/Git integration | ~$36 | https://1password.com/developers/students |
| digitalocean | DigitalOcean | $200 cloud credits, 1 year validity | $200 | https://www.digitalocean.com/github-students |

## Application Priority Guide

### Week 1: Foundation
1. GitHub Student Pack (gateway to 12+ services)
2. JetBrains (no Pack needed, instant)
3. Azure (no credit card, instant)
4. Figma Education (if design-related)
5. Notion Plus (immediate productivity boost)

### Week 2: After Pack Approval
6. GitHub Copilot (auto-enabled)
7. DigitalOcean $200
8. Frontend Masters 6 months
9. 1Password 1 year
10. MongoDB Atlas $50

### Week 3+: Everything Else
- Cloud credits (AWS, Google Cloud)
- Learning platforms (Coursera FA, Kaggle)
- Media discounts (Spotify, Apple Music)
- Remaining Pack benefits

### Total Estimated Value

| Category | Estimated Annual Value |
|----------|----------------------|
| GitHub Pack benefits | ~$2,200+ |
| Independent dev tools | ~$500+ |
| Cloud credits | ~$600+ |
| Design tools | ~$400+ |
| Productivity | ~$200+ |
| Learning (if paid) | ~$500+ |
| **Total** | **~$4,400+** |
